# Data Availability

## Included in this public repository

This curated repository includes:

- selected processed daily datasets
- compact TEA-related outputs and modelling tables
- selected remote-sensing support tables that are small enough for distribution here
- complementary spatial layers used for site context and masking
- final lightweight figures and result tables associated with the manuscript-facing workflow

## Not included in this public repository

The following categories are intentionally excluded:

- raw eddy covariance source packages
- large pixel-level predictor tables
- heavy intermediate raster shard outputs
- legacy/experimental modelling branches not required for the curated release
- internal reviewer workspace files
- manuscript Word files and tracked-change review assets

## Reason for exclusion

Some upstream inputs are excluded because of one or more of the following:

- file size constraints for a clean GitHub repository
- data sharing restrictions or unclear redistribution status
- lack of necessity for a manuscript-facing public release
- risk of confusing reproducibility with exploratory or legacy materials

## Requesting additional data

Raw eddy covariance and large model input datasets are not included in this repository due to file size and/or data sharing restrictions. Publicly shareable processed datasets, final figures, tables, scripts and workflow documentation are provided when available. Additional data may be requested from the corresponding author, subject to availability and permissions.
