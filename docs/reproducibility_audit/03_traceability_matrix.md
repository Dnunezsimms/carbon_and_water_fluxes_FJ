# Traceability Matrix

Status labels used here:

- `Reproducible`: input, script, and output path are reasonably identifiable
- `Partially reproducible`: major pieces exist but the final export path or exact script version is ambiguous
- `Not currently reproducible`: key script, input, or export step is missing or hardcoded

Confidence labels:

- `High`: strong file-level evidence
- `Medium`: plausible match, but some ambiguity remains
- `Low`: only partial evidence or competing candidates

## General Assessment

The repository already supports a meaningful level of manuscript traceability, especially for:

- annual flux summaries,
- TEA-derived water partitioning,
- the selected landscape NEE model,
- and the landscape mean NEE values reported in the revised manuscript.

The main remaining weakness is not lack of scientific material, but lack of a single declared canonical production path for:

- the site map,
- CART figure exports,
- supplementary tables,
- and the final distinction between current and legacy spatial modelling workflows.
