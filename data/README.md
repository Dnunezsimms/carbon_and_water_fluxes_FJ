# Data Directory

This folder contains the subset of datasets judged suitable for the curated public release.

## Included

- `processed/`: compact processed datasets used by manuscript-facing analyses
- `external/complementary/`: spatial support layers and small supporting files
- `external/remote_sensing_predictors/`: compact seasonal predictor tables used to trace the modelling workflow

## Excluded from this public repository

Not included here:

- large raw pixel-level predictor tables
- heavy intermediate exports
- full working-archive data branches that are not required for this curated release

See `../docs/data_availability.md` for the repository-level policy.
